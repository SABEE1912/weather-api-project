# Weather API Project

A Python-based Weather API application that fetches real-time weather data, validates the data using Regular Expressions, stores records in MySQL, and maintains validation logs.

## Features

- Live Weather Data Fetching
- MySQL Database Integration
- Regex-based Validation
- Validation Reports
- Validation Logging
- Raw API Response Storage
- Weather History Tracking
- Hottest and Coldest City Search
- Export Weather History
- Export Invalid Records
- Failed Validation Counter
- Validation Status Tracking (PASSED / FAILED)

## Technologies Used

- Python
- MySQL
- Requests
- Regular Expressions (Regex)
- Logging Module
- WeatherAPI
